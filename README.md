# Installation
[Install Hugo](https://gohugo.io/installation/)     
[Install Git](https://git-scm.com/)       
[Install Chocolately Package Manager](https://chocolatey.org/)      

# Start Application
1. Clone repo to local machine
2. Host the site locally:
```sh
$ hugo server
```
